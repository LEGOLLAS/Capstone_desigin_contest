API 오픈 API를 통해 애플리케이션을 제작할 수 있습니다.
기본정보
서비스명 소규모 행사 오픈 API
서비스 유형 REST
데이터포맷 JSON
메서드 GET
요청 URL 117.17.102.69:3000/json
요청 변수
요청 변수 타입 필수 여부 기본값 설명
id integer Yes auto_increment Id값으로 기본키로 쓰입니다.
name string Yes - 행사명입니다.
sponsor string Yes - 행사 주최자입니다.
start_date date Yes 0000-00-00 행사 시작 날짜입니다.
end_date date Yes 0000-00-00 행사 끝나는 날짜입니다.
time string Yes - 행사 시간입니다.
addr string Yes - 행사 장소입니다.
intro string Yes - 행사에 대한 설명입니다.
API 호출 예제
사용언어 Node.js 
          
            var http = require("http");
            var req = http.get({hostname:"117.17.102.69", port: 3000, path:"/json"}, res =>{
              var string = "";
              res.on('data', data=>{
                string += data.toString();
              })

              res.on('end', ()=>{
                var something = JSON.parse(string);
                console.log(something);
              })
            })
